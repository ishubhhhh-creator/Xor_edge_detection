# Xor_edge_detection
This hardware-centric project explores an optimized approach to digital image processing by implementing an Edge Detection algorithm through fundamental Bitwise XOR operations. Traditional edge detection methods, such as the Sobel or Canny filters, often require significant computational resources including multiplications and floating-point arithmetic; however, this project leverages the inherent properties of the XOR gate to detect rapid transitions in pixel intensity. By performing a bitwise comparison between a target pixel and its immediate horizontal and vertical neighbors, the logic identifies boundaries where logical discrepancies occur, effectively highlighting the "edges" of objects within a frame. Designed with hardware synthesis in mind, the architecture is tailored for low-power FPGA or ASIC environments where real-time performance and gate-count efficiency are prioritized over complex mathematical modeling. The project demonstrates a deep understanding of digital logic gates and their application in high-speed, hardware-accelerated computer vision.
